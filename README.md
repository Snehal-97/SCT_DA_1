# SCT_DA_1: Excel Sales & Trend Performance Dashboard

## 📌 Project Overview
This project presents an end-to-end interactive sales performance dashboard built using **Microsoft Excel** as part of the SkillCraft Technology Data Analytics Internship (Task 1).

Using retail transaction data (`train.csv`), the dataset was cleaned, structured, and summarized using **Pivot Tables**, **Pivot Charts**, **Slicers**, and **Custom Formatting** to evaluate product category performance and multi-year revenue growth.

## 🔑 Key Metrics & Performance Insights
* **Total Revenue Generated:** **₹2.26M** across 4,922 unique orders.
* **Top Performing Category:** **Technology** led overall revenue with **~₹827.5K**, followed closely by **Furniture** (**~₹728.7K**) and **Office Supplies** (**~₹705.4K**).
* **Multi-Year Growth:** Sales grew consistently from **2015 (~₹479.9K)** to a peak in **2018 (~₹722.1K)**, demonstrating strong long-term expansion.

---

## 🛠️ Excel Features & Techniques Applied
1. **Data Cleaning & Preprocessing:** Converted raw dataset into structured Excel Tables (`Ctrl + T`) and ensured date parsing accuracy.
2. **Dynamic Summaries with Pivot Tables:** Aggregated sales metrics by product category and order dates.
3. **Data Visualization:** Built formatted Clustered Column Charts for category distribution and Line Charts for temporal trends.
4. **Custom Currency Formatting:** Implemented custom format codes (`"₹"#,##0, "K"`) for clean readability across axis labels and data callouts.
5. **Interactive Dashboards:** Embedded multi-table connected **Slicers** (*Region* and *Segment*) for real-time dynamic filtering.
